Resumenes Semanales

Semana 1
Fundamentos de las tecnologias web

Los fundamentos de las tecnologías web se refieren a los conceptos y componentes básicos que conforman el entorno de desarrollo y funcionamiento de las aplicaciones web. Algunos de los fundamentos clave incluyen: HTML, CSS, JS, HTTP, URL, Servidores Web, Navegadores Web, Backend y Frontend.

Semana 2
Herramientas de desarrollo

Visual Studio Code es una herramienta versátil y potente que proporciona un entorno de desarrollo completo para programadores y desarrolladores. Es ampliamente utilizado en la comunidad de desarrollo de software debido a su flexibilidad, extensibilidad y soporte multiplataforma.

Semana 2
Lenguaje Marcado de Hipertexto, Hojas de Estilo

El lenguaje marcado de hipertexto (HTML, por sus siglas en inglés) y las hojas de estilo en cascada (CSS, por sus siglas en inglés) son dos tecnologías fundamentales para la creación y diseño de páginas web. Trabajan en conjunto para definir la estructura, el contenido y la apariencia visual de un sitio web.

Semana 3
Programación Front-End JavaScript

Se refiere al desarrollo de la parte visual e interactiva de un sitio web o aplicación web utilizando el lenguaje de programación JavaScript. JavaScript es un lenguaje de scripting que se ejecuta en el navegador web del cliente y permite la creación de experiencias interactivas y dinámicas en el lado del cliente.

Semana 4
Manejo del Doom con JavaScript

El manejo del DOM (Document Object Model) con JavaScript es fundamental para interactuar con los elementos HTML de una página web y realizar cambios dinámicos en tiempo real. Aquí se muestran algunas técnicas comunes para manejar el DOM utilizando JavaScript:

♥ Seleccion de elementos
♥ Modificación de contenido
♥ Modificación de atributos
♥ Manipulación de clases
♥ Manipulación de estilos
♥ Creación de nuevos elementos
♥ Uso de eventos

Semana 5
Desarrollo FrontEnd con Framework JS

El desarrollo Front-End con un framework JavaScript es una práctica común y eficiente para crear interfaces de usuario complejas y escalables. Los frameworks JavaScript proporcionan estructuras y herramientas que simplifican el desarrollo web al ofrecer una arquitectura organizada y componentes reutilizables.

Semana 6
Eventos, Condicionales y Bucles con Framework JS

Los eventos son acciones o sucesos que ocurren en la interfaz de usuario y a los que se puede responder mediante código. Los condicionales se utilizan para tomar decisiones basadas en una condición determinada. Los bucles permiten repetir una serie de instrucciones varias veces.

Semana 9
Enrutamiento y Consumo API

El routing o enrutamiento se refiere a la forma en que las aplicaciones web manejan las solicitudes de los usuarios y determinan qué acción tomar en función de la URL que se está solicitando. En el contexto del desarrollo web, el enrutamiento se utiliza para mapear las URL a diferentes componentes o controladores que procesarán y responderán a esas solicitudes.

Semana 10
Uso de Hooks

Los hooks son funciones especiales que permiten a los desarrolladores utilizar características de React y gestionar el estado de componentes funcionales sin necesidad de escribir una clase de componente.

Semana 11
Next.JS

Next.js es un framework de desarrollo web de código abierto basado en React, que se utiliza para construir aplicaciones web modernas y escalables. Proporciona una serie de características y beneficios que hacen que el desarrollo de aplicaciones web sea más eficiente y productivo.

Next.JS-II

Next.js es una herramienta poderosa para desarrollar aplicaciones web modernas con React, proporcionando una amplia gama de funcionalidades que mejoran la eficiencia y la productividad del desarrollo, al tiempo que garantiza un rendimiento y una experiencia de usuario óptimos.

API RESTFUL-SPRING BOOT

Spring Boot es un framework de desarrollo de aplicaciones basado en Spring que facilita la creación de aplicaciones Java, incluidas API RESTful.

Tecnologías Utilizadas

FIGMA

By: CPMI 234 123

Figma es una herramienta de diseño y prototipado basada en la nube que se utiliza para crear interfaces de usuario, diseños gráficos y prototipos interactivos. Es ampliamente utilizado por diseñadores de productos digitales, equipos de diseño y desarrolladores para colaborar en la creación de diseños de alta calidad y experiencias de usuario intuitivas.

HTML

By: CPMI 456 264

HTML es el lenguaje de marcado estándar, el cual utiliza etiquetas (tags) para definir elementos y estructuras en una página web. Estas etiquetas se utilizan para marcar y describir diferentes partes del contenido, como títulos, párrafos, encabezados, enlaces, imágenes, listas y muchos otros elementos. Cada etiqueta tiene un propósito específico y se utiliza para formatear y organizar el contenido de manera semántica.

CSS

By: CPMI 431 312

El lenguaje CSS utiliza reglas que consisten en una selección de elementos HTML y una serie de propiedades y valores. Las reglas se aplican a los elementos correspondientes en el documento HTML, lo que permite controlar su apariencia visual. Se pueden aplicar estilos de forma directa a elementos individuales o se pueden utilizar selectores para aplicar estilos a múltiples elementos a la vez.

JavaScript

By: CPMI 431 312

JavaScript es un lenguaje de programación de alto nivel, interpretado y orientado a objetos. Es ampliamente utilizado en el desarrollo web para agregar interactividad y funcionalidad a las páginas web.

JavaScript se ejecuta en el lado del cliente, es decir, en el navegador web del usuario. Permite realizar acciones y manipular elementos en la página web en respuesta a eventos, como hacer clic en un botón, mover el mouse o enviar un formulario. Además de su uso en el desarrollo web, JavaScript también se puede utilizar en entornos de servidor, aplicaciones de escritorio y dispositivos móviles.

React.JS

By: CPMI 431 312

React.js es una biblioteca de JavaScript de código abierto utilizado para construir interfaces de usuario interactivas y reutilizables. Fue desarrollado por Facebook y se ha vuelto muy popular en el desarrollo web.

Vite

By: CPMI 431 312

Vite es una herramienta de compilación rápida y eficiente que se utiliza habitualmente en el desarrollo de aplicaciones web con Vue.js y React.js. Proporciona un entorno de desarrollo ágil y optimizado, mejorando la productividad y la experiencia del desarrollador.
